FAQ sobre Comandos Git

1. O que é o Git?

Git é um sistema de controle de versão distribuído que permite acompanhar mudanças em arquivos e facilitar o trabalho colaborativo entre desenvolvedores.

2. Como inicializar um repositório Git?

Para iniciar um repositório Git, execute o seguinte comando no diretório do seu projeto:

git init

3. Como clonar um repositório Git?

Para clonar um repositório remoto, use o comando:

git clone

4. Como adicionar arquivos para o stage?

Para adicionar arquivos ao stage (preparar para commit), use o comando:

git add

Ou para adicionar todos os arquivos de uma vez:

git add .

5. Como fazer um commit?

Após adicionar arquivos, você pode realizar um commit com o seguinte comando:

git commit -m "Mensagem do commit"

6. Como enviar minhas alterações para o repositório remoto?

Para enviar suas alterações ao repositório remoto, use:

git push

7. Como atualizar meu repositório local com mudanças do repositório remoto?

Para atualizar o repositório local com as últimas alterações do remoto, use o comando:

git pull
